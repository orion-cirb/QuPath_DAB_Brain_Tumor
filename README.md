### Images description

2D images of sections of human brain, tumor cells stained with DAB

2. *DAB* channel: tumor cells

### Plugin description

* Detect tissue in RGB channel with a pixel classifier
* In tissue, detect tumor cells in DAB channel with a pixel classifier
* Give tumor cells number and area in each annotation provided

### Dependencies

* **QuPath pixel classifiers** named *tissue.json* and *DAB.json*
